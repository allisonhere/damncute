# Damn Cute Shortcodes

## [damncute_term_filters]
Renders taxonomy filter controls for the pet archive and taxonomy pages.

## [damncute_pet_meta]
Outputs a metadata row for a single pet, showing Age, Owner, and Adoption status when those fields are set.

## [damncute_pet_social]
Renders the owner/social profile block for a single pet, including a linked handle and reaction buttons/counters.

## [damncute_pet_most_loved]
Displays a grid of the most‑loved pets (sorted by total reactions) on single pet pages.

## [damncute_pet_related]
Shows related pets on single pet pages based on shared species/breed, vibe, or owner.

## [damncute_pet_sections]
Builds the narrative sections on a single pet page: Why are they so cute?, About, and Favorite snack.

## [damncute_pet_gallery]
Renders the pet’s gallery attachments (images/videos) from the `gallery` post meta on single pet pages.

## [damncute_submit_cta]
Outputs the submit CTA button or floating prompt (variant and label configurable via shortcode attrs).

## [damncute_pet_submit_form]
Embeds the Forminator submission form configured in Settings → General (Form ID stored in theme option).

## [damncute_pet_of_day]
Renders the “Pet of the Day” feature card, using the scheduled/selected pet ID with a fallback to a random pet.
